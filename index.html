Introduction

This document will walk you through the process of creating your first additive manufacturing project.

Basics Needed

The minimum basics required to follow this guide:

  • Prusa Mk3S+
  • PC with SD card reader
  • PrusaSlicer Software installed
  • Desired filament for project
  • Filament Cutters
  • Paper Towels

File Preparation
  • Start by downloading the model .STL file to a safe place on your PC.
  • Next we will open the PrusaSlicer software and open the file we downloaded.
  • Set the print quality setting to the desired choice.
  • Set the filament to the correct type to match what you are using.
  • Some basic notes for print settings:
    • Infill is how dense the part will be. Generally it is best to use the lowest value that will still support the top features of the part.
    • A brim will help to keep the part from lifting from the print bed, especially if it has sharp corners.
    • Layer height will affect the detail of the part as well as the printing time.
      • 0.05mm - Super Fine
      • 0.10mm - Fine
      • 0.15mm - Balanced
      • 0.20mm - Strong & Thick
      • 0.25mm - Draft
    • Perimeters are what give 3D printed parts their strength. Two is minimum for visual parts, three is standard, and up to six is for high strength parts.
Printer Preparation

Start by powering up your machine. Select the filament that you will be using, this should match what was selected in your print settings. Clip the end of the filament to make a 45 degree point for easy loading. Use the printer's menu to warm up the extruder to the proper temperature of your filament. As this is happening, clean the print bed with warm water and dish soap to remove finger oils from the plate. Dry the plate and do not touch again with your bare hands. Replace the plate back onto the printer and load the new filament into the hot extruder. Once the filament is loaded, you are ready to start printing your part. Load the SD card with the .gcode file into the printer, use the printers' menu to navigate to the correct file and hit start!

Completion!

In a short time you will have a completed part if all goes well.
